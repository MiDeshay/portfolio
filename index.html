Picross!

Picross is a logic based puzzle game made with JavaScript, Canvas, HTML5, and CSS3. In this game, a player attempts to use hints located on the top and left side of a 10x10 grid fill or cross out tiles to reveal a hidden picture. Some highlights of this project include its clean user interface, object oriented design, and systematic document manipuation.

Spacebook

Spacebook is a clone of the popular social media application Facebook made via the Ruby on Rails and React/Redux frameworks. Users can create accounts, upload and edit profile and background pictures, add other users as friends and create posts with optional pictures. Some highlights of this application include its visual likeness to Facebook, use of AWS S3 for cloud based media storage, and a search bar which can be used to find other users in the database.

Visit the Page See the Repo

HotSpot

HotSpot is a location based social media application made with MongoDb and React/Redux in which users can create and join groups then create and attend events within a group. Users can create accounts, create and join groups, view, edit, or attend events within a particular group, and visit show pages for groups and other users. Some highlights of this application include it being a team project in which I played the role of team lead, its use of the google maps API, and functional search bars which allow users to search for both other users and groups.
